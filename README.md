## 📌 Project Overview
This is a production-grade desktop enterprise utility application built using **Core Java** and the **Java Swing Framework**. The software provides retail or corporate businesses with an automated, lightweight, and zero-latency console to handle multi-category inventory tracking and transaction logging without requiring complex remote database dependencies.

---

## ✨ System Features
- **Dynamic Ledger Formulation:** Implements real-time mathematical grid computations to evaluate sub-totals across multi-inventory lines automatically.
- **Categorized Inventory Inputs:** Features independent structural panels for specialized product streams (Medical Supplies and Grocery Categories).
- **Automated Receipt Generation:** Features a live structural receipt terminal that builds structured invoices displaying customer meta-data (Name, Contact, Bill Number) alongside full cost metrics instantly.
- **Robust Field Validation:** Built-in exception sandbox frameworks that prevent program crashes against null inputs or character formatting errors during numeric entry.

---

## 🛠️ Technology Stack
- **Language Layer:** Java 8+ / Core Java
- **Graphic Interface Framework:** Java Swing (`javax.swing`) & Abstract Window Toolkit (`java.awt`)
- **Execution Workflow:** Event Dispatch Thread (`ActionListener` architecture)
- **Data Model:** Local Object Arrays & Standalone Collection Arrays
